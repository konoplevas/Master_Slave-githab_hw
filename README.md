### Master-Slave репликация
- **Направление:** Одностороннее (Master → Slave)
- **Запись:** Только на Master
- **Чтение:** На всех серверах
- **Конфликты:** Невозможны
- **Сложность:** Простая

### Master-Master репликация  
- **Направление:** Двустороннее
- **Запись:** На любой узел
- **Чтение:** На всех серверах
- **Конфликты:** Вероятны
- **Сложность:** Сложная
